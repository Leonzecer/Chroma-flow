A tiny utility for turning raw colors into human-readable color identities.

Designed for Stable Diffusion / Forge workflows where describing colors consistently becomes annoying.

## Why?
AI workflows often hit a weird bottleneck:

You know the color visually,
but not what to call it.

Chroma Flow converts sampled colors into readable identity labels like:

- scarlet color
- dusty lavender color
- sand color

Useful for:
- character consistency
- prompt building
- LoRA workflows
- color communication
- AI art iteration

## Current Features
- RGB color sampling
- nearest color identity matching
- XKCD color database support
- lightweight and simple workflow

## Status
Initial public release.
Built to solve a tiny but annoying workflow problem

## Installation

1. Open your Forge `extensions` folder

2. Clone this repository:

```bash
git clone https://github.com/Leonzecer/Chroma-flow.git
```

3. Restart Forge

4. Chroma Flow will appear inside the UI

## How to use

1. Pick a color from the RGB picker
2. Chroma Flow finds the closest readable color identity
3. Use the generated label inside prompts or workflow notes

Example outputs:
- scarlet color
- dusty lavender color
- sand color
